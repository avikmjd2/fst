# FST Based Morph-Analyser
## Intructions to run:
You need to clone the repo by this command

>  git clone https://github.com/avikmjd2/fst

Then when in your Vs-code:
<br>
If you are on WSL, run these commands in the repective braches of _fst/fst_ and _fst/fsl_
> python3 -m venv venv
> source venv/bin/activate
> pip install pynini
> python main.py
These steps will run the code
